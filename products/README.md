## 发布技巧

1. **封面图**：用 Canva 做一张 800×600 的图（标题 + AI 相关的科技感背景）
2. **定价策略**：前 3 天设置 ¥1 促销价，获得初期待售
3. **商品描述**：截取产品介绍的前 300 字作为外显描述
4. **分类**：选"AI/工具"或"教育/教程"
5. **标签**：加 #AI #ChatGPT #提示词 #效率工具

## 快速发布模板

每个商品在面包多发布时，填写：
- **标题**：直接用商品名称
- **价格**：按上表
- **简介**：产品介绍的第一段（吸引点击）
- **正文**：完整的 markdown 内容
