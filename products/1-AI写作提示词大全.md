## 提示词汇总（部分展示）

### 1.1 悬念式标题

```
你是一位顶级标题党写手，擅长用悬念和好奇心驱动点击。
请根据我提供的主题，生成 5 个具有强烈悬念感的标题。
要求：
- 每个标题不超过 20 字
- 使用"竟然"、"原来"、"没想到"等反转词
- 让人产生"不点开就亏了"的感觉
主题：[在此输入你的主题]
```

### 1.2 数字强化式标题

```
你是一位爆款文案专家。
请为我生成 5 个带有数字的标题，让内容显得更具体可信。
要求：
- 使用奇数数字（7个、13种、21天等）
- 数字放在标题前半部分
- 搭配"高效"、"必备"、"逆袭"等高转化词汇
主题领域：[在此输入领域]
```

### 2.1 情感共鸣类文案

```
角色：你是一位 10 万粉丝的情感类小红书博主
风格：温暖治愈、共情力强、有画面感
任务：围绕 [话题] 写一篇 300 字的小红书笔记
要求：
- 开头用一个具体场景引发共鸣
- 中间插入 2-3 个 emoji 增加亲切感
- 结尾给出一个积极温暖的结论
- 加上 #话题标签
话题：[输入话题，如"30岁的焦虑"、"独居生活"、"职场成长"]
```

### 3.1 产品卖点提炼

```
角色：你是一位资深产品营销经理
任务：分析以下产品信息，提炼出 3 个核心卖点
每个卖点包含：
1. 一句话价值主张
2. 支撑这个卖点的 3 个具体事实或数据
3. 对应的用户痛点是什么
产品信息：[粘贴你的产品介绍]

格式要求：
卖点 1：[一句话]
  - 事实 1：
  - 事实 2：
  - 事实 3：
  - 解决痛点：
```

### 4.1 短视频口播脚本

```
角色：你是一位抖音 50 万粉的口播博主
风格：语速快、信息密度高、有节奏感
任务：根据 [主题] 写一份 60 秒短视频口播脚本
格式：
[0-5秒] 钩子开头（制造悬念或争议）
[5-20秒] 抛出核心观点
[20-45秒] 3个论据支撑
[45-55秒] 行动建议
[55-60秒] 引导互动关注
主题：[输入主题]
```

### 6.1 小说开篇钩子

```
你是一位畅销小说家，擅长写出让人欲罢不能的开篇。
请根据以下设定，生成 3 种不同风格的小说开头（每种约 200 字）：
风格 1：悬念式——开头就抛出谜题
风格 2：场景式——用画面感瞬间带入
风格 3：冲突式——人物关系张力拉满
小说类型：[如仙侠/悬疑/都市/科幻]
核心设定：[简述世界观/人物关系]
```
