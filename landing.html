常见问题

快速了解 TriGen

TriGen 和其他平台有什么区别？ ▼
TriGen 聚合了 9 个主流 AI 平台、300+ 模型，你无需注册多个平台、管理多个 API Key。一个控制台即可调用所有的最新模型。
模型是实时可用的吗？ ▼
是的，所有模型都通过我们后端的实时 API 代理调用。我们支持自动切换备用渠道，即使某个平台故障，也能保证服务可用性。
积分如何计算和使用？ ▼
每个模型消耗不同数量的积分（0-25 分/次对话）。免费模型不消耗积分，高级模型（如 GPT-5、Claude 4 Opus）消耗更多。注册即送体验积分。
是否支持 API Key 接入？ ▼
支持！你可以在控制台的 API Key 管理中创建自己的 Key，然后通过 OpenAI 兼容接口调用 TriGen 的所有模型。
如何保障数据安全？ ▼
所有对话数据经过加密传输，我们不存储你的聊天记录。API Key 由你自行管理，也可选择使用平台提供的公共额度。
