🏗️ 分章创作

不要一次生成全文。每章单独创作，系统会自动携带：

📌 大纲约束

每章开头自动注入大纲要求：
✅ 本章应推进的情节节点
✅ 本章核心冲突/转折
✅ 关键人物出场/退场
⚠️ AI 可能偏航，发现问题及时修正

👤 人物一致

每章注入当前出场的角色档案：
✅ 性格/口头禅/外貌特征
✅ 当前状态/所在地点
✅ 人际关系/目标动机
长文中 AI 容易忘配角，记得核对

⚠️ 注意事项

❌ 字数精确：AI 不会数数，只能给大概范围（如"约3000字"）
❌ 剧情横跳：需大纲+摘要双重约束
💡 建议：每写5-10章回顾一次，手动修正偏离

❓ 常见问题

什么是 Meshy？为什么需要 API Key？▼
Meshy 是一个 AI 3D 内容生成平台，支持文本生成3D模型和图生3D。使用 Meshy 的 API 需要注册账号并申请 API Key，免费额度可参考官网说明。
生成的3D模型是什么格式？▼
Meshy 支持输出 GLB、FBX、OBJ、STL 等常见 3D 格式，适用于游戏引擎（Unity/Unreal）、3D 打印、AR/VR 等场景。
生成一个3D模型需要多长时间？▼
通常文本转3D需要 5-15 分钟，图生3D需要 10-30 分钟，具体取决于模型的复杂度和当前服务器负载。
我的 API Key 安全吗？▼
API Key 仅存储在您本地浏览器中（localStorage），不会被上传到任何服务器。请妥善保管，避免泄露。
